# TengeGuard
TengeGuard — веб-сервис для поиска и контроля подписок пользователя. Проект подключается к Gmail в режиме read-only, анализирует реальные письма с чеками, оплатами, trial-периодами и продлениями, показывает понятный дашборд подписок, историю отменённых сервисов и уведомляет о важных датах через Telegram-бота
